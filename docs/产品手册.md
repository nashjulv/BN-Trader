# BN-Trader v1.0.0 产品手册

## 1. 产品概述

BN-Trader 是一个面向小白的本地单机版短线交易系统，连接币安交易所，通过 5 种行情场景智能识别，辅助用户进行短线交易。

### 核心特性

| 特性 | 说明 |
|------|------|
| 智能场景识别 | 自动识别趋势/震荡/突破/反转/极端 5 种行情，自动匹配策略 |
| 资金池管理 | 仓位分配、资金回收、复利/反马丁格尔仓位调整 |
| 多层风控 | 单笔→日度→账户三层风控，超限自动阻止交易并强制复盘 |
| 双模式交易 | 手动下单 + 自动策略交易，一键切换 |
| 真实交易 | 支持币安 API Key，真实下单 + 自动同步账户余额 |
| 本地优先 | SQLite 本地存储，API Key 仅保存在本机 |
| 明亮/暗黑 | 默认浅色 Ant Design 风格，一键切换暗色主题 |

## 2. 安装与运行

### 系统要求

- Python 3.10+
- Windows 10+ / macOS 12+ / Linux (Ubuntu 20.04+)

### 安装步骤

```bash
# 1. 进入项目目录
cd BN-Trader

# 2. 创建虚拟环境
python3 -m venv .venv

# 3. 激活虚拟环境
source .venv/bin/activate     # macOS/Linux
# 或
.venv\Scripts\activate        # Windows

# 4. 安装依赖
pip install -r requirements.txt

# 5. 运行
python main.py
```

### 打包为独立安装程序

```bash
python package.py --clean        # 构建当前平台安装包
python package.py --platform win64  # 指定 Windows
```

## 4. 配置流程（首次使用）

### 4.1 配置币安 API

1. 登录 [币安官网](https://www.binance.com) → API 管理
2. 创建 API Key（**仅开启现货交易权限，切勿开启提现权限**）
3. 点击侧栏 **⚙** 或顶栏 **"API"** 进入 API 设置页
4. 填写 API Key / Secret Key → **"测试连接"** → 确认清单全部正常 → **"保存"**
5. 系统自动同步账户余额到主界面「资金概览」

API 设置页包含：
- **连接状态**：已连接/未连接、服务器时间、延迟、权限
- **连接测试清单**：API Key、Secret Key、服务器、权限、时间同步
- **安全提示**：本地存储说明与权限建议

### 4.2 设定资金 / 风控 / 持仓参数

分别点击侧栏 **◎ 资金**、**⬡ 风控**、**▣ 持仓** 进入对应设置页：

| 页面 | 分组 | 参数 |
|------|------|------|
| **资金设置** | 资金池 | 初始总资金、准备金比例、单笔最大仓位、日亏损上限 |
| **风控设置** | 单笔风控 | 单笔最大亏损%、单笔止盈%、最大持仓时间(秒) |
| | 日度风控 | 最多日交易次数、最大连续亏损、冷却时间(秒) |
| | 账户风控 | 最大回撤比例 |
| **持仓设置** | 仓位管理 | 仓位倍数 |
| | 止盈止损 | 移动止损%、最低盈亏比 |
| | 持仓管理 | 超时自动平仓(秒) |

点击 **"保存并应用"** 即刻生效；点击 **"恢复默认"** 恢复出厂值。

### 4.3 设定策略参数

1. 点击侧栏 **◈** 或顶栏 **"策略"**
2. 顶部 Tab 切换 5 种策略，分别设定参数：

| 策略 | 核心参数 | 默认状态 |
|------|---------|---------|
| 趋势策略 | 快/慢线周期、MACD、ADX、止损/止盈% | **启用** |
| 震荡策略 | 布林带周期、RSI 超买超卖线 | **启用** |
| 突破策略 | 回看周期、成交量倍数、突破阈值% | **启用** |
| 反转策略 | RSI 极值、背离检测窗口 | 禁用 |
| 极端策略 | ATR 倍数、成交量激增倍数 | 禁用 |

3. 勾选要启用的策略，调整参数，点击 **"保存并应用"**
4. 顶栏 **"策略"** 按钮变为绿色 ✓

### 4.4 启用自动交易

1. 确保 API 已配置 ✓、策略已配置 ✓
2. 点击顶栏 **"手动"** → 切换为 **"自动"**（红色）
3. 系统按当前行情场景调用对应策略生成信号并执行
4. 右侧下单面板自动锁定（禁用）

## 5. 功能详解

### 5.1 行情场景识别

| 场景 | 图标 | 特征 | 触发策略 | 建议仓位 |
|------|------|------|---------|---------|
| 趋势行情 | ↗ | ADX>25，均线多头/空头 | 趋势跟踪 | 30-50% |
| 震荡行情 | ⇋ | 布林带收窄，RSI 30-70 | 高抛低吸 | 20-30% |
| 突破行情 | ↑ | 价格突破关键位+放量 | 突破跟进 | 40-60% |
| 反转行情 | ↺ | 顶/底背离，RSI极端 | 逆势试探 | 10-20% |
| 极端行情 | ⚡ | ATR 飙升，成交量异常 | 反向或观望 | 5-10% |

场景可手动切换：点击主界面左栏「行情场景」中的快捷按钮。

### 5.2 资金池管理

```
总资金池
├── 可用资金      可直接用于交易的资金
├── 冻结资金      已下单未成交
├── 持仓保证金    当前持仓占用
└── 风险准备金    强制预留（默认20%）
```

**资金回收机制**：
- 盈利 → 按仓位倍数（连胜 +0.1x，最高 1.5x）加大下次仓位
- 亏损 → 按仓位倍数（连败 -0.15x，最低 0.5x）减小下次仓位

### 5.3 风控体系

| 层级 | 条件 | 触发后果 |
|------|------|---------|
| 单笔 | 止损>2%、持仓>1h | 拒绝下单 |
| 日度 | 日亏>5%、交易>20次、连亏3次 | 暂停交易+冷却300秒 |
| 账户 | 回撤>20%、准备金<10% | 禁止交易 |
| 强制 | 单笔大亏5%、连亏3次、日亏3% | **弹窗强制复盘** |

主界面「风控状态」以三根进度条实时展示：日亏损额度、仓位控制、账户回撤；超过 60%/80% 会变为警告色/危险色。

### 5.4 强制复盘

触发条件：连续亏损 3 次 / 日亏损 >3% / 单笔亏损 >5%

复盘表单包含：
1. 错误分析（必填）
2. 市场分析（选填）
3. 情绪状态选择
4. 改进计划（必填）
5. 经验教训（必填）

填写完毕提交后恢复交易权限。

### 5.5 手动下单

右侧「手动下单」面板：

1. 选择交易对
2. 切换 **限价单** / **市价单**（市价单禁用价格输入）
3. 输入价格、数量，或用 25% / 50% / 75% / 100% 快捷按可用余额填充
4. 可选填写止盈、止损
5. 点击 **买入 / 做多**（绿）或 **卖出 / 做空**（红）
6. 底部显示预估手续费与保证金

### 5.6 日志

**主界面迷你日志**（K 线下方）：
- 过滤：[全部] [交易] [风控] [系统]
- 操作：清空日志

**日志详情页**（侧栏 ☰）：
- 表格列：时间、类型、状态、内容
- 类型色标：交易=蓝、风控=橙、系统=灰
- 状态色标：成功=绿、警告=橙、失败=红
- 底部分页：上一页 / 下一页

### 5.7 主题切换

点击顶栏 **☀**（明亮）/ **☾**（暗黑）切换。默认启动为明亮主题。

| 主题 | 背景色 | 卡片色 | 强调色 | 适用场景 |
|------|--------|--------|--------|---------|
| 明亮（默认） | `#F5F7FA` | `#FFFFFF` | `#1890FF` | 白天交易 |
| 暗黑 | `#0D1117` | `#1C2333` | `#1890FF` | 夜间盯盘 |

所有面板（场景色、表格、进度条、K 线背景、侧栏）自动跟随切换。

## 7. 技术指标参考

| 指标 | 计算 | 用途 |
|------|------|------|
| MA (SMA) | N 周期收盘价简单平均 | 趋势方向判断 |
| EMA | N 周期指数加权平均 | 趋势跟踪（滞后更小） |
| MACD | 12EMA - 26EMA, 9EMA 信号线 | 金叉死叉信号 |
| RSI | 100 - 100/(1+RS), RS=平均涨幅/平均跌幅 | 超买>70, 超卖<30 |
| 布林带 | MA20 ± 2×σ | 波动率区间，触及上下轨 |
| ATR | N周期真实波幅均值 | 止损距离参考 |
| ADX | +DI/-DI 差值的移动平均 | 趋势强度(>25=趋势) |

所有指标实现在 `indicators/technical.py`，纯 NumPy 计算。

## 9. 常见问题

**Q: 没有币安 API Key 能用吗？**
A: 可以。行情数据（K 线、价格）无需 API Key 即可查看。账户余额同步和真实下单必须配置 API Key。

**Q: API Key 安全吗？**
A: 仅保存在本机 `~/.bn_trader_api.json`，不上传任何服务器。建议仅开启「现货交易」权限，关闭提现。

**Q: 自动交易会亏钱吗？**
A: 自动交易仅根据技术指标生成信号，不保证盈利。系统有多层风控（止损、日亏上限、强制复盘），但市场有风险，请谨慎使用。

**Q: 如何停止自动交易？**
A: 点击顶栏 **"自动"** 切换回 **"手动"** 即可。

**Q: 资金 / 风控 / 持仓参数怎么改？**
A: 点击左侧侧栏对应图标进入设置页，修改后点 **"保存并应用"**。

**Q: 数据保存在哪里？**
A:
- 数据库：`bn_trader.db`（项目目录）
- API Key：`~/.bn_trader_api.json`
- 策略参数：`~/.bn_trader_strategies.json`
- 全局参数：`~/.bn_trader_global.json`
- 日志：`bn_trader.log`（项目目录）

# BN-Trader v1.0.0 产品资料

> 本地短线交易与风险管理系统

## 产品定位

面向数字资产短线交易者的桌面工作台，将行情识别、资金管理、风险控制、下单和复盘集中在一个本地应用中。

## 目标用户

- 需要统一观察行情、仓位与风险状态的短线交易者
- 希望先用模拟模式验证策略，再谨慎接入实盘 API 的用户
- 重视交易纪律、止损与复盘流程的个人交易者

## 核心能力

- 五类行情场景识别：趋势、震荡、突破、反转与极端行情
- 手动交易与策略驱动的自动交易模式
- 资金池、单笔风险、日度风险和账户回撤管理
- 持仓、盈亏、交易日志与强制复盘
- Binance API 连接检测与本地凭据配置
- Apple 风格的明亮/暗色桌面界面

## 支持平台

- Windows 10/11 x64
- macOS Apple Silicon
- Python 3.10 及以上的源码运行环境

## 安全与隐私

- API Key 仅保存在本机，不写入仓库或帮助资料
- 建议仅启用现货交易权限，禁止提现权限
- 首次使用先保持手动或模拟模式，确认风控参数后再启用自动交易

## 推荐上线流程

1. 使用模拟或只读环境确认行情与账户数据。
2. 设置资金、风控、持仓及策略参数。
3. 用小额交易验证订单链路与日志。
4. 定期复盘，并根据真实风险承受能力调整限制。

## 风险声明

本产品是交易辅助工具，不构成投资建议，也不保证收益。数字资产价格波动剧烈，用户应独立判断并承担交易风险。

## 1.1 项目介绍

### 什么是BN-Trader？

BN-Trader是一个本地短线交易系统，专为数字货币交易新手设计。它连接币安交易所，提供智能行情分析和自动交易功能。

### 核心功能

- **智能场景识别**：自动识别5种行情场景（趋势、震荡、突破、反转、极端）
- **资金池管理**：智能仓位分配，资金回收与再投资
- **多层风控**：单笔/日度/账户三层风控保护
- **强制复盘**：亏损达到阈值后强制暂停交易并要求复盘
- **自动/手动双模式**：既支持策略自动交易，也支持手动操作

### 为什么用Python？

```python
# Python代码示例：计算RSI指标
def rsi(data, period=14):
    """相对强弱指数 - 衡量超买超卖状态"""
    deltas = data[1:] - data[:-1]  # 计算价格变化
    gains = sum(d for d in deltas if d > 0) / period  # 平均涨幅
    losses = -sum(d for d in deltas if d < 0) / period  # 平均跌幅

    rs = gains / losses  # 相对强度
    return 100 - (100 / (1 + rs))  # RSI公式

# 这就是一个完整的RSI指标计算函数，仅需6行代码！
```

Python的优势：
- 语法简洁，适合编程初学者
- 有丰富的数据处理库（NumPy, Pandas）
- 有成熟的GUI框架（PyQt6）
- 币安官方推荐Python开发

## 1.2 环境搭建

### 步骤1：检查Python版本

打开终端，运行：
```bash
python3 --version
# 输出应为 Python 3.10 或更高版本
```

### 步骤2：创建虚拟环境

虚拟环境可以隔离项目依赖，避免不同项目之间的冲突：
```bash
cd BN-Trader
python3 -m venv .venv
```

### 步骤3：激活虚拟环境

**macOS / Linux:**
```bash
source .venv/bin/activate
```

**Windows:**
```bash
.venv\Scripts\activate
```

激活后，终端提示符会显示 `(.venv)`。

### 步骤4：安装依赖

```bash
pip install PyQt6 numpy pandas SQLAlchemy requests websocket-client pyqtgraph python-dotenv pytest
```

或者使用项目提供的安装脚本：
```bash
bash install.sh
```

## 1.6 关键概念

### config.py 的作用
- 所有可调参数集中在一个地方
- 修改参数不需要改代码
- 方便做A/B测试

### 虚环境的作用
- 每个项目独立依赖，互不影响
- `pip list` 只会看到当前项目的包
- 退出虚拟环境：`deactivate`

### 为什么分为 core/models/services
- **core**：应用生命周期管理、数据库连接
- **models**：数据结构定义（表结构）
- **services**：业务逻辑处理
- **gui**：用户界面显示

这种分层设计的好处：
- 修改界面不影响业务逻辑
- 替换数据库不影响界面
- 每个模块可以独立测试

## 常见问题

**Q: pip install 失败怎么办？**
A: 先尝试 `pip install --upgrade pip`，如果还是失败，检查网络连接或使用国内镜像源。

**Q: 为什么提示 "command not found: python"？**
A: macOS 上默认使用 `python3`，或者创建别名：`alias python=python3`

**Q: 虚拟环境激活后，pip 还是装的全局包？**
A: 检查终端提示符是否有 `(.venv)` 前缀，或者使用 `which pip` 确认使用的是虚拟环境中的 pip。
